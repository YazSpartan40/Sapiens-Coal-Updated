# Charcoal Mod for Sapiens

## Overview
The Charcoal mod introduces efficient charcoal fuel production to Sapiens! Transform wood into superior charcoal fuel that burns 5x longer than regular branches, making it perfect for extended cooking, smelting, and heating sessions.

## How to Use

1. **Learn Campfire Cooking** skill (required prerequisite)
2. **Gather branches or logs** from trees
3. **Approach a campfire or kiln** and open the crafting menu
4. **Select a charcoal recipe** and craft it
5. **Use the resulting charcoal** as fuel in any fire-based object

## Strategic Benefits

- **Early Game**: Convert branches into charcoal for much longer campfire sessions
- **Mid Game**: Use logs to produce charcoal for efficient kiln operations
- **Late Game**: Stockpile charcoal for extended smelting and cooking operations

## Compatibility
- **Hammerstone not required** - works with vanilla Sapiens
- Fully compatible with other mods
- No performance impact

## Installation
1. Download the mod files
2. Place the `Charcoal` folder in your Sapiens mods directory
3. Enable the mod in the world creation menu
4. Start a new world or load an existing save
